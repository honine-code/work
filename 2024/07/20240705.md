# Campain Manager Function
## ConfigManager
1. 휴일 생성
- PathVariable: 고객사KEY
- ReqBody: 휴무일자, 휴무내용
- Res: 생성된 휴일정보
2. 휴일 수정
- PathVariable: 고객사KEY, 휴일KEY
- ReqBody: 변경할 날짜, 변경할 내용
- Res: 변경된 휴일정보
3. 휴일 삭제
- PathVariable: 고객사KEY, 휴일KEY
- Res: 삭제된 휴일정보
4. 휴일 리스트 조회
- PathVariable: 고객사KEY
- Parameter: 휴무일자, 휴무내용
- ResL 휴무일자, 휴무내용
5. 발신번호 리스트 응답
- PathVariable: 고객사 키, 서비스유형(workcenter)
- Res: 발신번호 리스트
